## 🛠️ 기술 스택

**Language |** Java 11

**Framework |** Spring Boot 2.7.9

**Database |** AWS RDS(MySQL), H2, Redis 

**ORM |** JPA(Spring Data JPA), QueryDSL

**CI/CD |** Github Action, Docker

**Test |** Junit5, Mockito, H2

**Docs |** Swagger
